# tidehunter

A bot which analyzes the volume of any spot trading pairs on Binance and sends alerts on Telegram on any abnormal spike in the volume.

## Features
- Subscribe via Telegram API on any trading pairs
- Set the timeframe to subscribe to, e.g. 1min or 1hour
- Set the delta of the volume spike before an alert is sent
- Grant or revoke admin or normal access to any users
